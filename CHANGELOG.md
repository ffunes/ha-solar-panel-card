# Changelog

## v1.0.0

Initial release.

- Real-time solar panel array with per-panel power visualization (green fill bar).
- String mode (central inverter, total power divided across panels) and microinverter mode (one sensor per panel).
- Automatic W/kW detection.
- Optional per-panel names (`panel_names`).
- `blank` placeholder to skip positions in the grid.
- Visual configuration editor with entity validation.
- Click a panel to open its more-info dialog.
